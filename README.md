# TensorTools
A simple repository to manage my Python ML projects.

# Usage
`pip install tensortools`

```python
import tensortools as tt
tt.ops.flatten([[1, 2], [2, 3]])  # returns [1, 2, 3, 4] as a tf.Tensor
```
